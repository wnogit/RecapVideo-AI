## 1ï¸âƒ£ API Key Rotation (Backend)

### Current Problem
- Only uses primary key, no fallback/rotation

### Solution
Add round-robin rotation for Gemini and Transcript API keys.

### Database Changes
Already have `usage_count` in `api_keys` table.

### Logic
```python
# Get least-used active key for load balancing
async def get_rotated_key(key_type: str) -> str:
    keys = await db.execute(
        select(APIKey)
        .where(APIKey.key_type == key_type, APIKey.is_active == True)
        .order_by(APIKey.usage_count.asc())  # Least used first
        .limit(1)
    )
    key = keys.scalar_one_or_none()
    if key:
        key.usage_count += 1
        return key.key_value
    return None
```
