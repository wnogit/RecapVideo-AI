# Admin Settings Redesign - API Keys & Configuration Management

## Overview

This document outlines the redesign for the Admin Settings page with a focus on API key management, payment configuration, and Telegram integration. The new design supports **multiple keys per service**, full **CRUD operations**, **search/filter**, and **usage tracking**.

---

## ğŸ¯ Goals

1. **Multiple API Keys per Service** - Support rotation, fallback, and load balancing
2. **Full CRUD Operations** - Add, View, Edit, Delete for each key
3. **Search & Filter** - Quickly find keys across all services
4. **Usage Tracking** - See which keys are being used and how often
5. **Security** - Masked display with reveal option, primary key designation
6. **Telegram Integration** - Bot token and admin IDs for notifications
